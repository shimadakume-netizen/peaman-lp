Features

4つのモードで
全部解決する

ヒアリングから投稿作成・画像生成・週次ToDoまで、収益化に必要な機能をすべて搭載。

01
HEARING MODE

🎤 ヒアリングモード

現状・悩みを聞き出し、Threads×note運用の「素材シート」を自動作成。次のステップに必要な情報を整理します。

  • 現状・目標のヒアリング
  • 素材シート自動生成
  • 次のアクション整理
02
WRITING MODE

✍️ 執筆モード

素材シートからThreads投稿・note記事を自動生成。SNSマーケティング4ステップに沿った構成で仕上げます。

  • Threads投稿文の自動生成
  • 4ステップ構成で記事執筆
  • 素材シート作成後に利用可能
03
IMAGE MODE

🎨 図解モード（画像生成）

Threads投稿用の画像生成プロンプトを自動作成。ChatGPT/Geminiに貼るだけで完成。

  • 16:9（Threads用）/ 縦長（壁紙用）対応
  • 雰囲気・カラー・スタイルをカスタマイズ
  • AIに貼るだけで即完成
04
TODO MODE

📋 Todoモード（週次）

6フェーズ診断で「今自分がどこにいるか」を特定し、今週やるべき具体的なToDoを自動生成。

  • 4ステップ×6フェーズで現在地診断
  • 1日2〜3タスク×7日の実行計画
  • 毎週更新で無理のない継続
