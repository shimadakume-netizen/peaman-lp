Features

7つのモードで
全部解決する

相談から記事執筆・図解・AIアート・ロードマップ・ToDoまで、収益化に必要な機能をすべて搭載。

01
CONSULT MODE

💬 何でも相談

記事作成・収益化プロセスについて何でも相談できる窓口。「次はどうすれば？」に具体的に答えます。

  • 「何を書いたらいい？」という相談
  • 各モードの使い方説明
  • SNSマーケの知識に基づくアドバイス
02
HEARING MODE

🎤 記事ヒアリング

あなたの考えやテーマを聞いて、記事の方向性を決定。「何を書くか」から「どう書くか」まで一緒に考えます。

  • 「何について書きたいか」をヒアリング
  • どんな構成にするか一緒に考える
  • テーマ・ターゲット・メッセージを明確化
03
WRITING MODE

✍️ 記事本文作成

ヒアリングの内容をもとに、あなたの言葉とスタイルを活かした高品質な記事を生成します。

  • Threads向け短編（140文字〜）
  • note向け記事（1000〜5000文字）
  • あなたの言葉・スタイルを活かした執筆
04
DIAGRAM MODE

📐 図解作成

Threads投稿やnoteで使える図を作成。データを視覚的にわかりやすく表現します。

  • 画像生成AIのプロンプト作成
  • Mermaid図など複数形式で出力
  • データを視覚的にわかりやすく表現
05
AI ART MODE

🎨 AIアート

Midjourneyプロンプト設計アシスタント。「何を作りたいか」を対話形式で整理してコピペ用プロンプトを自動生成。

  • 初心者向けステップバイステップガイド
  • コピペで使える英文プロンプト自動生成
  • 雰囲気・カラー・スタイルをカスタマイズ
06
ROADMAP MODE

🗺️ ロードマップ作製

「何をしていいかわからない」をビジュアルで解決。現在地とゴールを整理して次のステップを明確化。

  • Threads×note収益化の全体像を把握
  • ビジュアルロードマップ画像を生成
  • 現在地とゴールの整理・次ステップ明確化
07
TODO MODE

📋 Todo作成

収益化フェーズを診断して今週のタスクを自動生成。SNSマーケの4ステップ×6フェーズで現在地を特定します。

  • 4ステップ×6フェーズ診断
  • 実行可能な週次タスク生成
  • ビジュアルロードマップ画像を生成
